# tmux

## Shortcuts

| Key | Pane management |
| :--- | :--- |
| ^b Up | select pane above |
| ^b ^Up | resize up |
| ^b E | resize evenly |
| ^b ! | pane to new win |
| ^b " | vertical split |
| ^b % | horizontal split |
| ^b x | kill |
| ^b z | zoom |

| Key | Win management |
| :--- | :--- |
| ^b 0 | select win 0 |
| ^b & | kill |
| ^b c | create |

| Key | Session management |
| :--- | :--- |
| ^b $ | rename session |
| ^b s | choose session |

| Key | Misc |
| :--- | :--- |
| ^b \[ | copy mode |
| ^b |  |
